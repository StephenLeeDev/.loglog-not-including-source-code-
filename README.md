# .loglog

It's a portfolio about Flutter project .loglog that I created
<br>It's a Japanese commercial Social Media project that introduces nearby shops and discount coupons
<br>It links between shops and customers

<br>It is just my portfolio repository so it doesn't include source code
<br>It contains only screenshots, descriptions, and demonstrations<br><br><br>

- MVVM design pattern
- Clean Architecture
- Provider
- GetIt
- Dio
- GoRouter
- Secure Storage
- Shared Preferences
- ImagePicker
- Open Street Map
- Table Calendar
- REST API integration
- Firebase AppDistribution

<br><br>
